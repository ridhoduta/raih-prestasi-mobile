# 📘 UI Design Guidelines -- Aksen Hijau Profesional

Versi: 1.0\
Platform: Flutter (Mobile)\
Fokus: Tampilan modern, clean, dan profesional dengan aksen hijau.

------------------------------------------------------------------------

## 🎨 1. Sistem Warna (Color System)

Gunakan **1 primary color** dan variasinya untuk menjaga konsistensi
visual.

### Primary Palette

-   **Primary Green**: `#2E7D32`
-   **Light Green Background**: `#E8F5E9`
-   **Accent Green**: `#66BB6A`
-   **Background Base**: `#F9FAFB`
-   **Text Primary**: `#212121`
-   **Text Secondary**: `#757575`

### Rules:

-   Jangan gunakan hijau terang berlebihan.
-   Gunakan hijau sebagai aksen, bukan dominasi.
-   Background utama harus netral (off-white / putih lembut).
-   Gunakan kontras tinggi untuk readability.

------------------------------------------------------------------------

## 🧱 2. Card Component Guidelines

### Border Radius

-   Gunakan radius 16--20 untuk tampilan modern.

### Shadow

-   Gunakan soft shadow dengan opacity rendah (5--8%).
-   Hindari shadow gelap dan keras.

### Spacing

-   Padding internal card: 16px
-   Jarak antar card: 16--20px

### Image

-   Gunakan rasio 16:9
-   Rounded hanya di bagian atas
-   Tambahkan gradient overlay tipis jika perlu

------------------------------------------------------------------------

## 🏷 3. Badge / Chip Design

Gunakan soft background (bukan warna solid terang).

Contoh: - IT → background hijau sangat muda - Nasional → amber lembut -
Aktif → light green + text primary green

### Rules:

-   Font size: 11--12
-   Font weight: Medium
-   Padding horizontal: 10--12
-   Radius: 30 (pill style)

------------------------------------------------------------------------

## 🔘 4. Button Design

### Primary Button

-   Background: Primary Green
-   Text: White
-   Border Radius: 14--16
-   Elevation: 2--3
-   Padding vertical: 14--16

### Rules:

-   Jangan gunakan warna gradasi berlebihan.
-   Gunakan full-width untuk CTA utama.
-   CTA harus paling menonjol di dalam card.

------------------------------------------------------------------------

## 🔤 5. Typography Hierarchy

### Judul Kompetisi

-   Font Size: 18--20
-   Font Weight: SemiBold (600)

### Deskripsi

-   Font Size: 14
-   Warna: Text Secondary

### Informasi Tambahan (Tanggal, Status)

-   Font Size: 12--13
-   Warna: Text Secondary lebih redup

### Rules:

-   Maksimal 2--3 ukuran font berbeda dalam satu card.
-   Gunakan line height 1.3--1.5 untuk kenyamanan baca.

------------------------------------------------------------------------

## 📱 6. Bottom Navigation

-   Background: White
-   Active Icon: Primary Green
-   Inactive Icon: Grey 400
-   Tambahkan subtle top border (opacity rendah)

### Rules:

-   Jangan gunakan background hijau penuh.
-   Label active boleh sedikit lebih tebal.

------------------------------------------------------------------------

## 🔍 7. Search Bar

-   Background putih
-   Border radius 16--20
-   Shadow sangat halus
-   Icon warna abu gelap

### Rules:

-   Jangan gunakan border tebal.
-   Gunakan hint text warna abu soft.

------------------------------------------------------------------------

## 📐 8. Spacing System

Gunakan sistem spacing konsisten:

-   Small: 8
-   Medium: 12--16
-   Large: 24

### Rules:

-   Jangan gunakan angka acak.
-   Gunakan multiple dari 4 atau 8.
-   Pastikan alignment rapi secara vertikal.

------------------------------------------------------------------------

## ✨ 9. Prinsip Profesional UI

1.  Konsisten warna
2.  Hierarki visual jelas
3.  Spacing rapi
4.  Kontras cukup
5.  Tidak berlebihan animasi
6.  Gunakan white space

------------------------------------------------------------------------

## 🎯 Goal Akhir

UI harus terlihat: - Modern - Clean - Tidak seperti template default -
Siap production - Profesional dengan aksen hijau elegan
